# Команды для терминала

Выполни эти команды в директории `c:\Users\wiwal\GIT\Resume\portfolio`:

## 1. Установка зависимостей
```powershell
npm install
```

## 2. Создание папки для проектов (опционально)
```powershell
New-Item -ItemType Directory -Force -Path public/projects
```

## 3. Запуск dev-сервера
```powershell
npm run dev
```

Сайт будет доступен на http://localhost:3000

## 4. Для production build (позже)
```powershell
npm run build
npm start
```
